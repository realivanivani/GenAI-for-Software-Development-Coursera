## Course 1: Introduction to Generative AI for Software Development

### Module 1: Foundations of Generative AI
This module covers the basics of generative AI, including fundamental concepts, algorithms, and tools used in the field.

- **Topics Covered:**
  - Introduction to Generative AI
  - Key Differences Between Generative and Discriminative Models

  - Evaluating Generative Models
